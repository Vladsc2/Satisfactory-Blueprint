# Satisfactory Blueprint

The project was created while playing Satisfactory, as a replacement for drawing factory schemes in Paint. But the project grew a bit and supports automatic resource calculation, power consumption calculation, factory ramp-up simulation, comments, saving factory layouts, etc... About half of the game's content is implemented. The project is currently abandoned

**Important:** The project displays correctly only at 1920×1080 resolution and 100% UI scaling

## Installation and launch

Python version
1) Download the repository as a `.zip` file and extract it
2) Open a console and navigate to the project folder
3) Create a virtual environment: `python -m venv .venv`
4) Activate the environment: `.\.venv\Scripts\activate` (Windows)
5) Install dependencies: `pip install -r requirements.txt`
6) Run: `python main.py`

Exe version
1) Download the repository as a `.zip` file and extract it
2) Run `SF Blueprints.exe`
